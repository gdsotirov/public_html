Libraries

DNetMod - DeviceNet Module

My diploma project tha I continue supporting. It's a C++ library using the C API of DeviceNet™ interfaces. The goal of the project is to simplify the use of DeviceNet™ interfaces by providing a universal way of communication between your application, the interfaces and the devices in a DeviceNet™ network.

  • License: GPL v2
  • Repository: gdsotirov/dnetmod
  • Bug reports: Sotirov-bg.net Bugzilla
  • Technologies used: C++, HTML4, CSS, Shell, Perl
  • Development environment: KDevelop
  • Download: GitHub
